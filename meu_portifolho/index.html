Sobre mim

Sou desenvolvedor fullstack junior com experiência em diferentes plataformas. No frontend, trabalho com HTML, CSS e JavaScript para criar interfaces funcionais. No backend, uso Python com Django para APIs e sistemas web robustos.

Além do desenvolvimento web, também desenvolvo aplicações mobile e desktop usando Python com Kivy - uma stack que me permite criar apps multiplataforma de forma eficiente. Para banco de dados, trabalho com SQLite em projetos menores e MySQL para aplicações mais complexas.

O que mais me motiva é a versatilidade de poder criar desde uma página web até um app mobile ou sistema desktop, sempre buscando soluções práticas e código limpo. Estou sempre estudando para evoluir como desenvolvedor.
