Sobre mim

Sou um desenvolvedor fullstack junior apaixonado por tecnologia e sempre em busca de novos desafios. Minha jornada começou com curiosidade sobre como as coisas funcionam "por trás das telas", e hoje estou transformando essa paixão em uma carreira sólida no desenvolvimento de software.

Atualmente estou cursando Técnico em Desenvolvimento de Sistemas no SENAC, onde aprofundo meus conhecimentos em programação. Tenho certificações da Cisco Networking Academy em Python e C#, além de certificações Oracle em banco de dados. No frontend trabalho com HTML, CSS e JavaScript, e no backend minha especialidade é Python com Django.

O que mais me motiva é a versatilidade que posso criar uma página web até aplicações mobile e desktop com Kivy. Como desenvolvedor junior, estou sempre estudando, praticando e buscando evoluir minhas habilidades para entregar soluções cada vez melhores.
